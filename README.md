# WhatsApp Bulk Sender Bot (Python + Selenium)

A lightweight and customizable automation tool that sends bulk WhatsApp messages using Python and Selenium WebDriver.  
This bot interacts with **WhatsApp Web** through browser automation, allowing you to send text messages to a list of phone numbers or contacts with minimal effort.
